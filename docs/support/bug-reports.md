# Bug reports

Phenofhy is a beta package. Functions and error handling are still being refined, so
you may encounter breaking changes or inconsistent behaviors.

## Where to report issues

Please open a GitHub issue in the Phenofhy repository:

https://github.com/studiovincentstraub/phenofhy/tree/main

## How to open an issue

1. Open the repository link above.
2. Go to the Issues tab.
3. Click New issue.
4. Add a short title and include the details below.

## What to include

- Phenofhy version or commit
- Steps to reproduce
- Expected vs actual behavior
- Any error messages or logs
